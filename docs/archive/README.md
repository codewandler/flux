# docs/archive

Preserved, not maintained. Material here is kept for provenance but is **no longer canonical** — it
is finished, superseded, or a point-in-time snapshot. For the current picture see the
[docs map](../README.md).

- **`designs/`** — designs whose work has **landed**; the shipped behavior is documented in the living
  docs. (e.g. `bash-replacement.md` — generic `bash` is now opt-in with dedicated ops.)
- **`notes/`** — historical working notes and finished backlogs (`next-2026-06.md`, `improvements.md`,
  `dogfood-notes.md`).
- **`research/`** — a competitive-landscape and system-prompt **snapshot** from project inception;
  useful as reference, not kept up to date.

Active designs live in [`../designs/`](../designs/); current work lives on the
[story board](../stories/README.md).
